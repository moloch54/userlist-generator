# userlist-generator  
generate userslist with name, forname. Ex: john jmith -> j.smith,john-j ....

# Usage  
your userslist must be in this format, first name and last name separated with a space:  

john doe  
albert camus  

```sh
python usergen.py --ul users.txt -o userslist.txt  
```
